# ChatGPT Plus 账号白嫖指南

1. 安装谷歌辅助自动化扩展(仅第一次需要安装配置)
   github: [codex-oauth-automation-extension](https://github.com/luck-hao/codex-oauth-automation-extension.git)

   git 拉取代码后，打开 Chrome 扩展页面，启用开发者模式，加载已解压的扩展，选择 `codex-oauth-automation-extension` 目录。
   ![图1](https://scm-image.yiheda.com/yhd-service-app-version/paypal/image.png)
   ![图2](https://scm-image.yiheda.com/yhd-service-app-version/paypal/image-1.png)

2. 配置codex2api中转站接入(仅第一次需要配置)
   a. 打开扩展侧边栏，选择来源为 `codex2api`
   ![配置来源](https://scm-image.yiheda.com/yhd-service-app-version/paypal/image-2.png)
   b. codex2api 填入 `http://10.14.0.113:8888/admin/accounts` 作为 API Host，保存设置。
   c. 管理密钥填入 `634e4abae463aaf5ff8d621e59ad46ac`
   ![alt text](https://scm-image.yiheda.com/yhd-service-app-version/paypal/image-3.png)
3. 配置邮箱注册 (仅第一次需要配置)
   a. 邮箱服务选择QQ邮箱, 点击旁边的登录,登录自己的QQ邮箱账户
   ![alt text](https://scm-image.yiheda.com/yhd-service-app-version/paypal/image-4.png)
   ![alt text](https://scm-image.yiheda.com/yhd-service-app-version/paypal/image-5.png)
   b. 邮箱生成选择 `DuckDuckGO`
   ![alt text](https://scm-image.yiheda.com/yhd-service-app-version/paypal/image-6.png)
   c. 点击注册邮箱最右边的获取按钮配置DuckDuckGo邮箱绑定QQ邮箱接码, 第一次需要授权DuckDuckGo访问QQ邮箱(见截图), 授权后会自动获取验证码并填入注册邮箱的验证码输入框
   ![alt text](https://scm-image.yiheda.com/yhd-service-app-version/paypal/image-8.png)
   ![alt text](https://scm-image.yiheda.com/yhd-service-app-version/paypal/image-9.png)
   ![alt text](https://scm-image.yiheda.com/yhd-service-app-version/paypal/image-10.png)
   ![alt text](https://scm-image.yiheda.com/yhd-service-app-version/paypal/image-11.png)

   ![alt text](https://scm-image.yiheda.com/yhd-service-app-version/paypal/image-14.png)
4. 运行自动化 (必须开启日本节点代理)
   1. 点击启动, 步骤 1 - 5 完全不用管, 等到第六步时需要停止自动化，手动完成后续操作
      ![alt text](https://scm-image.yiheda.com/yhd-service-app-version/paypal/image-15.png)
5. 运行到第六步暂停
   1. ![alt text](https://scm-image.yiheda.com/yhd-service-app-version/paypal/image-16.png)
